сщИНСТРУКЦИЯ ПО ЗАПУСКУ САЙТА SENIOR ADVISORS
============================================

Этот файл содержит инструкции по локальному запуску сайта на вашем компьютере.

ВАРИАНТ 1: Открыть файлы напрямую в браузере
---------------------------------------------
1. Откройте папку "SA website"
2. Найдите файл "index-ru.html"
3. Дважды кликните на него (откроется в браузере по умолчанию)

ИЛИ щелкните правой кнопкой мыши → Открыть с помощью → выберите браузер (Chrome, Firefox, Edge)

ВАЖНЫЕ СТРАНИЦЫ:
- Главная: index-ru.html
- Портфолио: portfolio.html
- Клиенты: clients.html
- Команда: team.html
- Услуги: service-*.html

ПРИМЕЧАНИЕ: Некоторые функции могут не работать при прямом открытии файлов.


ВАРИАНТ 2: Запуск локального сервера (Рекомендуется)
-----------------------------------------------------

Если у вас установлен Python:

1. Откройте командную строку (Win + R → введите "cmd" → Enter)

2. Перейдите в папку с сайтом:
   cd "путь\к\папке\SA website"

   Например:
   cd "C:\Users\User\SA website"

3. Запустите сервер:
   python -m http.server 8000

4. Откройте браузер и перейдите по адресу:
   http://localhost:8000/index-ru.html

5. Для остановки сервера: нажмите Ctrl+C в командной строке


ТРЕБОВАНИЯ:
-----------
- Современный браузер (Chrome, Firefox, Edge, Safari)
- Для варианта 2: Python 3.x (проверить: python --version)
